QR Code Generator
  
Простой генератор QR‑кодов с графическим интерфейсом на PyQt6.

Как пользоваться:

  Введите текст или ссылку в поле ввода.   
  Нажмите кнопку Start — QR‑код появится в окне.  
  Нажмите кнопку Save.  
  
  В открывшемся окне выберите:
      папку для сохранения;  
      имя файла;  
      формат (PNG или JPG).  
      Нажмите «Сохранить» (или Save).  
Интерфейс:
    Текстовое поле — для ввода данных (текст, URL и т. д.).    
    Кнопка Start — запускает генерацию QR‑кода.    
    Область просмотра — показывает готовый QR‑код.  
    Кнопка Save — сохраняет QR‑код на компьютер.
